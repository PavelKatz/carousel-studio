Запусти стартовый визард carousel-studio через skill `carousel-setup`. Проведи пользователя по шагам: бренд и папки → референсы (разбор картинок в дизайн-токены через `extract-brand-tokens`) → голос бренда (`voice.md`) → опциональные генеративные фоны → пробный рендер. Веди по одному шагу за раз.

Если пользователь что-то уточнил в команде, учти это: $ARGUMENTS
