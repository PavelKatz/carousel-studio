# Стиль-референс: clean-editorial (пример)

Нейтральный пример, чтобы показать форму референса. **Без брендинга** — копируй и правь под себя, или заведи свой через `extract-brand-tokens`.

## Вид
Тёмный модерн-эдиториал: почти-чёрный фон, светлый текст, один мягко-голубой акцент (`#6ea8fe`) для плашек и CTA, тёплый второй акцент (`#f4a261`) для kicker'ов. Заголовки — гротеск (Inter 800), цитаты — курсивный serif (Lora). Спокойно, чисто, много воздуха.

## Палитра
- `bg #11141b` — фон слайдов
- `ink #07090d` — глубокий тёмный (текст на плашке)
- `fg #eef1f6` — основной текст
- `accent #6ea8fe` — плашки, CTA, акцент-глифы
- `accent2 #f4a261` — kicker'ы
- `muted #9aa3b2` — подзаголовки, номера страниц
- `positive #56d364`

## Шрифты (все зашиты в движок, кириллица + латиница)
- заголовки/тело/списки — **Inter**
- цитаты/подзаголовки — **Lora** (курсив)
- eyebrow/kicker/номера — **IBM Plex Mono**

## Как сделать своим
1. Поменяй `color.*` на свои hex (минимум `bg`, `fg`, `accent`).
2. Подбери шрифты (если нужен не из зашитых — добавь в `scripts/carousel-render/scripts/fetch-fonts.mjs` и `npm run fonts`).
3. Добавь лого в `motifs/` и пропиши `layout.logo` (если нужно).
4. Проверь пробным рендером (см. README движка).
