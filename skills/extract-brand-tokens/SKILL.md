# extract-brand-tokens

Разбирает картинку-референс на **дизайн-токены** для движка `${CLAUDE_PLUGIN_ROOT}/scripts/carousel-render/`. Зрение — твоё собственное (читаешь изображение через Read), без внешних сервисов.

Куда пишем (из `carousel.config.json` → `brandDir`, `owner`):
- `<brandDir>/<owner>/references/<id>/tokens.json` — машинные токены,
- `…/<id>/style.md` — человекочитаемое описание,
- копия исходника в `…/<id>/source/`, лого/декор в `…/<id>/motifs/`,
- запись в `<brandDir>/<owner>/references/_index.json`.

## Процесс

1. **Прочитай изображение(я)** через Read. Несколько → один стиль; извлекай общее.
2. **Пройди чек-лист** (ниже). Цвета — сэмпли реальные пиксели, не «на глаз».
3. **Согласуй `id` и `category`** с пользователем (категории — в `_index.json`).
4. **Запиши файлы.** Структуру `tokens.json` бери из:
   - схемы — `${CLAUDE_PLUGIN_ROOT}/examples/brand/_schema/tokens.schema.json`,
   - примера — `${CLAUDE_PLUGIN_ROOT}/examples/brand/acme/references/clean-editorial/tokens.json` (+ `style.md`).
5. **Скопируй исходник** в `source/`, лого/мотивы — в `motifs/`.
6. **Добавь запись** в `_index.json` (id, title, category, mood, use_when, channels, tokens). Если это первый референс — поставь его в `default`.
7. **Провалидируй** пробным рендером:
   `node ${CLAUDE_PLUGIN_ROOT}/scripts/carousel-render/src/cli.mjs --slides ${CLAUDE_PLUGIN_ROOT}/examples/sample-slides.json --tokens <новый tokens.json> --out /tmp/ref-check --scale 1` — и глазами проверь PNG.
8. **Верни** сводку: палитра (hex), шрифты + уверенность, что попало в плашки/лого/мотивы.

## Чек-лист извлечения

**Canvas.** Соотношение. Для IG — 1080×1350 (4:5). Безопасные поля прикинь по отступам текста (обычно 80–120px на 1080-канвасе).

**Палитра (точный hex).** Сэмпли пиксели. Роли: `bg`, `ink` (тёмный), `fg` (текст), `accent` (главный акцент/плашка), `accent2` (второй), `positive`/`negative` если есть, `muted`. 4–8 цветов.

**Типографика по ролям.** Семейство (гипотеза + **обязательный web-fallback** + уверенность high/medium/low), размер px на 1080-канвасе, насыщенность, регистр, трекинг, курсив. Роли: `eyebrow`, `kicker`, `title`, `subtitle`, `body`, `quote`, `listItem`, `cta`, `pageNum`.
- **Кириллица/языки:** если контент не латиница, а угаданный шрифт её не поддерживает (Cinzel, Cormorant, многие дисплейные latin-only) — подбери аналог с нужными глифами и отметь подмену в `style.md`. Зашитые в движок и покрывающие кириллицу+латиницу: **Playfair Display, Lora, Inter, IBM Plex Mono**. Другой шрифт → добавь в `scripts/carousel-render/scripts/fetch-fonts.mjs` и прогони `npm run fonts`.

**Раскладка.** Выравнивание, радиус скруглений, трактовка выделения ключевых слов (плашка/подчёркивание/нет) + её цвета.

**Лого.** Есть ли, угол, размер, на каких слайдах (`showOn`). Файл — в `motifs/`.

**Картиночный стиль.** `imagery.style` (1–2 фразы) + `imagery.promptStyle` (затравка для генеративных фонов).

**Мотивы.** Повторяющийся декор (россыпь иконок, watermark-буквы, нумералы, рамки) — id + назначение; ассеты в `motifs/`.

## Важно
- Цвета — пиксельно точные hex.
- У каждого шрифта — fallback + уверенность; кириллица/язык решены явно.
- Не выдумывай мотивы, которых нет.
- Финал — валидный по схеме `tokens.json` + пробный рендер.
